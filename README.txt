Инструкция по запуску для Windows:

0. Откройте файл config-sample.ini с помощью блокнота и введите
в поле phone и username ВАШИ данные, к которым привязан Ваш аккаунт в телеграм.

phone = +123456789
username = MyUserName

1. Установите Python c официального сайта: https://www.python.org/downloads/
или через магазин приложений Windows.
2. Войдите в командную строку
3. Введите путь к папке с программными файлами: cd ваш_путь
Например: cd C:\Users\Alexander\Desktop\WORK\
4. В коммандной строке введите команду: python ChannelUsers.py
!!Если не срабатывает - см. примечание!!
5. Введите номер, привязанный к вашему аккаунту в Telegram
6. Введите код, полученный от официального аккаунта Telegram в ваших сообщениях
7. Введите ссылку на канал, данные о пользователях которого Вам необходимо получить.
!!Вы должны обладать правами администратора!!
8. Дождитесь выполнения программы (это занимает до 1-2 минут).
В конце выполнения программы появится строка ввода с адресом директории.
Полученные данные сохраняются в папку проекта в виде файлов: 
- user_data.xlsx
- user_data.txt
- user_data.json

Примечания:
Если Вы установили python, но команда не выполняется:
1. Закройте командную строку.
2. Перейдите в папку, куда был установлен python.
Как правило, адрес имеет следующий вид: C:\Python39
3.Скопируйте адрес папки.
4.Перейдите: панель управления -> свойства системы -> Переменные среды
-> Системные переменные.
Выделите Path и нажмите Изменить.
Если в появившемся окне нет адреса папки с Python, то нажмите Создать,
после чего вставьте скопированный адрес и нажмите ОК.
5. Откройте командную строку Windows и повторите действия, начиная с п.3
